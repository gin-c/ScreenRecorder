本项目来自：
https://github.com/yrom/ScreenRecorder

基于此添加了<font color="red"><b>锁屏录屏</b></font>功能。

同时支持<b>无声模式</b>和<b>有声模式</b>。


注意事项：
首次使用会请求各种权限，如果没有同意，会导致功能无法正常使用。
其中有：

## 悬浮窗权限

- 保证可以锁屏录制

## 文件读写权限

- 保证可以保存录屏

## 录音权限

- 保证可以录音

## 录屏权限

- 保证可以录屏

## <font color="red">不允许</font>电量优化
- 防止在锁屏上10秒之后<b>无法录屏</b>
